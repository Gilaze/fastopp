Quick Start Guide

1

Initialize

python oppman.py migrate init

Initialize Alembic in your project

2

Add Models

Edit models.py

Add new models to your models.py file

3

Create Migration

python oppman.py migrate create "Description"

Generate migration files

4

Apply

python oppman.py migrate upgrade

Apply migrations to database

Basic Commands
